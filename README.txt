This directory contains

Python files and jupyter notebooks to produce the results from the associated paper

All the large files (PTA Data files, ceffyl data files, libraries, MCMC chain text files, GP prediction npz files) can be downloaded from
https://zenodo.org/records/19019126


First, you would need to install holodeck (follow GitHub instructions)
Then, you need to replace the python files in the holodeck directory with the files in teh folder here (python_files_to_replace)
Then, you need to re-run steps 3 and 4 from Installation section on https://github.com/nanograv/holodeck

After this, you can start using the notebooks and python files from this directory.

Keep in mind that you would need to install some more packages within that environment. You can find out which ones you need as you start using the notebooks.
